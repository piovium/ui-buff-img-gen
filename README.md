# UI Buff Image Generator

一个用于生成类七圣召唤风格的状态 UI 图标的工具。

## 安装与部署

### 安装依赖

```bash
bun install
```

### 开发模式

```bash
bun run dev
```

## 颜色预设

工具内置了 10 种颜色预设，分别对应常用的状态类型：

1. **通用状态**
2. **增益效果**
3. **减益效果**
4. **冰元素**
5. **水元素**
6. **火元素**
7. **雷元素**
8. **风元素**
9. **岩元素**
10. **草元素**

每个预设都包含完整的背景和前景颜色参数配置，可在界面中直接选择使用。

## 支持的图标

### 图像格式

- **PNG** (`.png`)
- **WebP** (`.webp`)

### 图像来源

- **URL** - 可输入完整url自动获取图像
- **DIR** - 可通过本地文件路径选择

### 通道数

- **单通道图像** - 采用唯一通道作为图标
- **3通道图像** - 合并3通道为单通道灰度图像作为图标
- **4通道图像** - 仅采用第4通道（透明度通道）作为图标

### 其他

- 推荐图标具有清晰的边缘和阶梯化的灰度
- 推荐尺寸为 100x100 像素或更高
- 推荐保留 20 像素的边缘留白

## 常用参数说明

- **Alpha 缩放指数** (`a_factor`) - 通过幂函数重新映射 Alpha 通道的灰度，控制灰度映射的曲线形状
  - 值越小，中间灰度越亮；值为1时为线性映射；值越大，中间灰度越暗
- **非主体区域 色相偏移** (`fg_h_offset`) - 非主体区域的色相偏移量（-127~127），用于调整图标中非高亮区域的色相与总前景色相的偏移
  - 偏离0越远，与主体对比越明显，差值超过25会有明显的色差
- **反色** - 反转 Alpha 通道，将透明区域变为不透明，不透明区域变为透明
- **羽化** - 对 Alpha 通道应用半径为2像素高斯模糊，在此算法影响下会产生宽越3像素的高饱和度描边效果
